Iconos

Iconos de Bootstrap 5

Uso básico

Puedes colocar iconos en casi cualquier lugar utilizando el prefijo CSS y el nombre del icono. Los iconos están diseñados para ser utilizados con elementos en línea (nos gusta la etiqueta para la brevedad, pero usar un es más semánticamente correcto). fa<i><span>

<i class="fas fa-camera-retro"></i>

Puedes hacer que todos tus iconos tengan el mismo ancho para que puedan alinearse verticalmente fácilmente, como en una lista o en un menú de navegación.

Agregue una clase de en el elemento HTML que hace referencia a su icono para establecer uno o más iconos en el mismo ancho fijo. Esto es genial para usar cuando variar el ancho de los íconos (por ejemplo, un ícono alto pero delgado encima de un ícono ancho pero corto) alteraría la alineación vertical. Para mayor claridad en el siguiente ejemplo, hemos agregado un color de fondo en el icono para que pueda ver el ancho fijo y también hemos aumentado el tamaño de fuente del elemento principal.fa-fw
